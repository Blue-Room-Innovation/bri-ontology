# 20. Glosario

- DPP: Digital Product Passport.
- OWL: Lenguaje de Ontologías Web (modelado semántico).
- SHACL: Lenguaje de constraints para grafos RDF.
- EPCIS: Estándar GS1 para eventos de trazabilidad.
- UNTP: Marco ONU/CEFACT de trazabilidad.
- Scorecard: Conjunto de indicadores (emisiones, circularidad).
- Stub: Declaración mínima de una IRI externa para reducir warnings.
