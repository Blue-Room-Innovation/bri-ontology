# 20. Glosario

- DWP: Digital Waste Passport (credencial verificada sobre residuos).
- DMWP: Digital Marpol Waste Passport (variante regulatoria MARPOL).
- WastePassport: Estructura interna (credentialSubject) con detalle del residuo.
- Waste / MarpolWaste: Entidad residuo (genérico / regulado MARPOL).
- ResidueInformation: Bloque con códigos y cantidades de un residuo concreto.
- OWL: Lenguaje de ontologías Web.
- SHACL: Reglas de validación RDF.
- EPCIS: Estándar GS1 para eventos de trazabilidad (movimientos / entregas).
- UNTP: Marco ONU/CEFACT (verifiable credential / product passport patterns).
- Codelist: Esquema SKOS con códigos controlados (tipo residuo, puertos, países).
- Stub: Declaración mínima de IRI externa para reducir warnings.
