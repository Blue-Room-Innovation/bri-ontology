# Glosario

- DPP: Digital Product Passport.
- OWL: Web Ontology Language, modelado semántico.
- SHACL: Shapes Constraint Language, validación de grafos RDF.
- EPCIS: Estándar GS1 para captura de eventos.
- UNTP: Marco de la ONU/CEFACT para trazabilidad.
