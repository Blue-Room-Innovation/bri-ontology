# ¿Qué es DPP (Digital Product Passport)?

Un Pasaporte Digital de Producto es un conjunto de datos estandarizados que acompaña al producto durante su ciclo de vida (fabricación, uso, mantenimiento, reciclaje). Facilita trazabilidad, transparencia y cumplimiento normativo.

Este repositorio ofrece:
- Ontología OWL/RDF para modelar productos y eventos.
- Shapes SHACL para validar pasaportes.
- Scripts y ejemplos para validar y evolucionar el modelo.

Sigue con `quick-validation.md` para ver algo funcionando rápido, o con `../concepts/data-model.md` para entender el modelo.
