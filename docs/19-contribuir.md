1. Crear rama descriptiva (feat/marpol-quantity, fix/shapes-range, chore/rename-artifacts).

4. Actualizar documentación afectada (modelo, shapes, glosario).
5. Pull Request con motivación y resultados (p.ej. mejora validación residuo gaseoso).
